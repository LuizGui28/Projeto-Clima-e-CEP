 # Clima e CEP - Consulta de clima + endereço
Projeto de aplicação web para consultar informações de clima em tempo real e dados de endereço atráves do CEP, consumindo APIs externas.

 # Funcionalidades: 
- Consulta de clima por cep digitado 
- Mostrar temperaturas e informações meteorológicas 
- Consulta de endereço pelo CEP

 # Tecnologias aplicadas: 
- HTML 
- CSS 
- JavaScript 

 # APIs: 
- OpenWeather (clima) /
- ViaCEP (CEP)


Entre no link do site: https://integrandoapi.netlify.app
